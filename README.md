# MOVIE STREAMING API

This is a Movie Management API built with the use use of Spring Boot, Spring Data JPA, Spring Security, MySQL.
  It is a RESTful backend for managing movie records having full CRUD operations, with an Integrated image upload, binary data storage using MultipartFile,
  Secured endpoints with Spring Security and implemented role-based access control and Migrated from HTTP Basic Authentication to database-driven authentication for better scalability.
